# 🏏 Cricket Match Win Predictor

A machine learning project that predicts the probability of a team winning an IPL (Indian Premier League) cricket match.  
Built with Python, Pandas, Scikit-learn, and Streamlit/Gradio for interactive UI.

---

## 📌 Features
- Predicts the winning probability of an IPL team in real-time.  
- Uses historical IPL dataset for training the model.  
- Provides insights like most influential features (venue, toss, batting order, etc.).  
- Interactive web app for easy user input and live prediction.  

---

## 📊 Dataset
- Dataset used: **IPL matches dataset** (available on Kaggle).
- Includes match results, venues, teams, toss decisions, and scores.

---

## ⚙️ Tech Stack
- **Python**  
- **Pandas, NumPy** – Data preprocessing  
- **Matplotlib/Seaborn** – Data visualization  
- **Scikit-learn** – Model training & evaluation  
- **Streamlit/Gradio** – Web app deployment
